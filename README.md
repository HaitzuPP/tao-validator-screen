# TAO Institute — Primary Subnet Validator Screen

Analysis of 9 Bittensor validators ahead of standing up a primary subnet validator. Each is assessed on three questions:

1. Purpose, and whether it only validates its own chain.
2. Owner self-stake vs outside nominator stake.
3. Staking fees (take) and commission earned.

Live site: https://haitzupp.github.io/tao-validator-screen/

Data sources: taostats API (on-chain stake, take, rewards) and tao.com API (validator identity, APR, dominance). Snapshot 2026-07-03.
